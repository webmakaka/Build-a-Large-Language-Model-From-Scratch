### 3.3.1. Простой механизм самовнимания без обучаемых весов

Цель механизма самовнимания — вычислить контекстный вектор для каждого входного элемента, который объединяет информацию из всех остальных входных элементов.

<br/>

<img src="../img/chapter03-pic01.png" alt="Простой механизм самовнимания без обучаемых весов">

Контекстный вектор вычислим ближе к концу параграфа.

<br/>

Текст, был преобразован во вложенные токены.

```python
import torch
inputs = torch.tensor(
  [
  [0.43, 0.15, 0.89],  # Your     (x^1)
  [0.55, 0.87, 0.66],  # journey  (x^2)
  [0.57, 0.85, 0.64],  # starts   (x^3)
  [0.22, 0.58, 0.33],  # with     (x^4)
  [0.77, 0.25, 0.10],  # one      (x^5)
  [0.05, 0.80, 0.55]   # step     (x^6)
  ]
)
```

<br/>

Входной вектор округлили в меньшую сторону, чтобы было понятно. Но вычисления продолжаем с использованием сотых.

<br/>

```python
inputs = torch.tensor(
  [
  [0.4, 0.1, 0.8], # Your    (x^1)
  [0.5, 0.8, 0.6], # journey (x^2)
  [0.5, 0.8, 0.6], # starts  (x^3)
  [0.2, 0.5, 0.3], # with    (x^4)
  [0.7, 0.2, 0.1], # one     (x^5)
  [0.0, 0.8, 0.5]  # step    (x^6)
  ]
)
```
